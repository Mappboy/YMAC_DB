Data Kiosk Overview

The Data Kiosk will allow users to easily access, view, transform, input and download spatial data here.

It is for the most part a wrapper around FME server.

If you feel comfortable using FME server, or would just like to learn speak to Cameron and he will enable access.

If you are unable to find a workbench or have an idea for a new work bench please contact Cameron.
